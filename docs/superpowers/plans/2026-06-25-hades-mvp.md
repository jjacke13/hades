# hades MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

## Context

`hades` is a new C++ AI-agent harness that ports the **MOOS-IvP** robotics architecture to a software agent: a central pub/sub **Blackboard** (the "session"), pluggable **Modules** (tools, chat, LLM), and a behavior-arbiter **Helm**. The design spec is committed at `docs/superpowers/specs/2026-06-25-hades-agent-harness-design.md`; the dev environment (Nix flake, `nix develop`, pinned `nixos-26.05`) is set up and verified (gcc 15.2 C++20, cmake 4.1, `libcpr`+`nlohmann_json`+`gtest` compile+link OK).

This plan builds the **MVP vertical slice** (spec §8): a terminal chat agent that (1) holds a conversation through a **configurable LLM provider** (any OpenAI-compatible endpoint via env-var key), (2) calls **one isolated subprocess tool** (`fs-read`), and (3) **hard-vetoes / confirm-gates** destructive or over-budget actions. It proves the whole architecture end-to-end: pub/sub decoupling, module lifecycle, the pluggable arbiter, isolated tools, and replayable observability. Deferred (post-MVP): v2 scoring arbiter, memory retrieval, sub-agent fan-out, monitor TUI, full phase state-machine.

**Outcome:** `nix develop --command ./build/hades manifests/dev.hades` opens a REPL you can chat with; it can read a file via the tool; `rm -rf`-style actions are blocked pending confirmation; the whole run replays from `session.log` via the `scope` CLI.

**Goal:** Ship the MVP chat agent on the Blackboard + Module + Arbiter architecture.

**Architecture:** Everything flows through one in-process `Blackboard` (latest-value map + append-only `Eventlog`). A single-threaded loop (`Blackboard::pump()`) drains a FIFO of posts and dispatches to subscribing Modules — no module calls another directly. The `Arbiter` runs the per-turn loop (ask LLM → veto-filter → act); `LLMModule` calls any OpenAI-compatible endpoint via `cpr`; `ToolRunner` runs tools as sandboxed subprocesses. Tools are isolated; core modules are synchronous (threading deferred).

**Tech Stack:** C++20 · CMake + Ninja · Nix dev shell (`nix develop`, `nixos-26.05`) · `libcpr` · `nlohmann/json` · GoogleTest · posix fork/exec + rlimit.

## Global Constraints

- **C++20**, stdenv `g++` 15.2. **Every build/test command runs inside `nix develop`** (e.g. `nix develop --command ctest --test-dir build`).
- **Build:** CMake ≥3.20 + Ninja, out-of-tree `build/` (gitignored). `find_package(cpr/nlohmann_json/GTest REQUIRED)`. nixpkgs attr is `libcpr` but the CMake target is **`cpr::cpr`**.
- **Immutability:** never mutate in place; `post()` a new Entry; build a fresh per-turn snapshot.
- **Many small files** (~200–400 lines, 800 max), one responsibility each.
- **Config = plain-text MOOS-style blocks** (`Section = name { key = value }`, `#` comments). No TOML/JSON config dep.
- **Secrets:** API key from a **named env var** (`api_key_env`); never inline; **redact** in the Eventlog; never log the key.
- **TDD:** RED → GREEN → COMMIT, every task. Commit style `<type>: <desc>` (feat/fix/refactor/docs/test/chore); no attribution footer.
- **Fail fast & visible** on bad config (`MalConfig`); never silently swallow errors.

## MVP simplifications (scoped on purpose — not gaps)

- **Single-threaded event loop** (queue + `pump()`), not threads-per-module — deterministic & testable; threading deferred. Tools still run as **isolated subprocesses**.
- **Arbiter v1 only** (LLM decides; objectives veto/gate). v2 scoring deferred.
- **`MODE` is a stub** (`"EXECUTING"`); hierarchical phases deferred.
- **One native tool** (`fs-read`) + a **minimal real MCP adapter** (stdio JSON-RPC `tools/call`); MCP tool-discovery deferred.

### Blackboard message keys
| key | payload | producer → consumer |
|---|---|---|
| `USER_MESSAGE` | string | Chat → Arbiter |
| `LLM_REQUEST` | `{messages, tools, model}` | Arbiter → LLM |
| `LLM_RESPONSE` | `{text, tool_call?, prompt_tokens, completion_tokens, stop_reason}` | LLM → Arbiter |
| `TOOL_REQUEST` | `{id, tool, args}` | Arbiter → ToolRunner |
| `TOOL_RESULT` | `{id, ok, content}` | ToolRunner → Arbiter |
| `NEXT_ACTION` | `{kind, ...}` | Arbiter (decision record) |
| `CONFIRM_REQUEST` | `{id, prompt, action}` | Arbiter → Chat |
| `CONFIRM_RESPONSE` | `{id, approved}` | Chat → Arbiter |
| `ASSISTANT_MESSAGE` | string | Arbiter → Chat |
| `BUDGET_SPENT_USD` | number (cumulative) | LLM → StayOnBudget |
| `MODE` | string (stub `"EXECUTING"`) | Arbiter |

```cpp
#include "hades/eventlog.h"
#include <fstream>
namespace hades {
Eventlog::Eventlog(std::string path) : path_(std::move(path)) {}
void Eventlog::add_redaction(std::string s){ if(!s.empty()) secrets_.push_back(std::move(s)); }
const std::vector<Entry>& Eventlog::entries() const { return entries_; }
static std::string redact(std::string s, const std::vector<std::string>& secrets){
  for (const auto& sec : secrets) {
    for (auto pos = s.find(sec); pos != std::string::npos; pos = s.find(sec, pos))
      s.replace(pos, sec.size(), "***REDACTED***");
  }
  return s;
}
void Eventlog::append(const Entry& e){
  entries_.push_back(e);
  std::string val = redact(e.value.dump(), secrets_);
  std::string line = std::to_string(e.ts) + "\t" + e.key + "\t" + e.source + "\t" + val + "\n";
  if(!path_.empty()){ std::ofstream f(path_, std::ios::app); f << line; }
}
}  // namespace hades
```

```cpp
#include "hades/blackboard.h"
#include "hades/eventlog.h"
#include <chrono>
#include <deque>
namespace hades {
namespace { double mono(){ using namespace std::chrono;
  return duration<double>(steady_clock::now().time_since_epoch()).count(); } }

struct Sub { std::string pattern; Handler h; double min_interval; std::map<std::string,double> last; };

struct Blackboard::Impl {
  Eventlog* log; double t0; std::uint64_t seq=0;
  std::map<std::string,Entry> latest;
  std::vector<Sub> subs;
  std::deque<Entry> queue;
};
// NOTE: declare `std::unique_ptr<Impl> p_;` in the header's private section.

static bool match(const std::string& pat, const std::string& key){
  if(pat=="*") return true;
  if(!pat.empty() && pat.back()=='*') return key.compare(0, pat.size()-1, pat, 0, pat.size()-1)==0;
  return pat==key;
}
Blackboard::Blackboard(Eventlog* log) : p_(std::make_unique<Impl>()) { p_->log=log; p_->t0=mono(); }
double Blackboard::now() const { return mono() - p_->t0; }
void Blackboard::subscribe(const std::string& pattern, Handler h, double mi){
  p_->subs.push_back({pattern, std::move(h), mi, {}});
}
void Blackboard::post(const std::string& key, nlohmann::json value,
                      const std::string& source, const std::string& aux){
  Entry e{key, std::move(value), source, aux, now(), ++p_->seq};
  p_->latest[key] = e;
  if(p_->log) p_->log->append(e);
  p_->queue.push_back(std::move(e));
}
std::optional<Entry> Blackboard::get(const std::string& key) const {
  auto it=p_->latest.find(key); if(it==p_->latest.end()) return std::nullopt; return it->second;
}
std::size_t Blackboard::queued() const { return p_->queue.size(); }
void Blackboard::pump(){
  while(!p_->queue.empty()){
    Entry e = p_->queue.front(); p_->queue.pop_front();
    for(auto& s : p_->subs){
      if(!match(s.pattern, e.key)) continue;
      double last = s.last.count(e.key) ? s.last[e.key] : -1e18;
      if(s.min_interval>0 && e.ts-last < s.min_interval) continue;
      s.last[e.key]=e.ts;
      s.h(e);
    }
  }
}
}  // namespace hades
```

## Task 3 — Manifest parser (MOOS-style blocks)

**Files:** Create `include/hades/config.h` (contract), `src/config/manifest.cpp`, `tests/test_manifest.cpp`. Modify CMake.

**Grammar:** `#` to EOL is a comment. A block is a header (`Section` or `Section = name`) optionally followed by a `{ ... }` body. Body entries are `key = value` (split on FIRST `=`; value may contain spaces; keys lower-cased), one per line; the one-line inline form `Section = name { key = value }` carries a **single** entry. A header with no braces (`Module = llm`) is a block with empty `kv`. Unknown/malformed lines append to `warnings` (never throw).

**Interfaces — Produces:** `parse_manifest`, `Manifest::session/of`, `Block`, `set_*_on_string`.

```cpp
#include <algorithm>

#include <sstream>
namespace hades {
static std::string trim(std::string s){
  auto ns=[](int c){return !std::isspace(c);};
  s.erase(s.begin(), std::find_if(s.begin(), s.end(), ns));
  s.erase(std::find_if(s.rbegin(), s.rend(), ns).base(), s.end());
  return s;
}
static std::string lower(std::string s){ for(auto&c:s)c=(char)std::tolower((unsigned char)c); return s; }
static void split_kv(const std::string& line, Block& b, Manifest& m){
  auto eq=line.find('=');
  if(eq==std::string::npos){ m.warnings.push_back("bad config line: "+line); return; }
  b.kv[lower(trim(line.substr(0,eq)))] = trim(line.substr(eq+1));
}
static void parse_header(std::string h, Block& b){
  auto eq=h.find('=');
  if(eq==std::string::npos){ b.section=trim(h); b.name=""; }
  else { b.section=trim(h.substr(0,eq)); b.name=trim(h.substr(eq+1)); }
}
Manifest parse_manifest(const std::string& text){
  Manifest m; std::istringstream in(text); std::string raw;
  Block* open=nullptr;
  while(std::getline(in, raw)){
    if(auto h=raw.find('#'); h!=std::string::npos) raw=raw.substr(0,h);
    std::string line=trim(raw);
    if(line.empty()) continue;
    if(open){                                   // inside a {...} body
      if(line=="}"){ open=nullptr; continue; }
      split_kv(line, *open, m); continue;
    }
    if(line=="{"){ if(!m.blocks.empty()) open=&m.blocks.back(); continue; }
    // header (possibly with inline braces)
    auto ob=line.find('{');
    Block b;
    if(ob!=std::string::npos){
      parse_header(line.substr(0,ob), b);
      auto cb=line.find('}', ob);
      std::string body = (cb==std::string::npos)? line.substr(ob+1) : line.substr(ob+1, cb-ob-1);
      m.blocks.push_back(b);
      if(!trim(body).empty()) split_kv(trim(body), m.blocks.back(), m);
      if(cb==std::string::npos) open=&m.blocks.back();   // body continues next lines
    } else {
      parse_header(line, b); m.blocks.push_back(b);       // header-only or pre-brace
    }
  }
  return m;
}
std::optional<Block> Manifest::session() const {
  for(auto& b: blocks) if(lower(b.section)=="session") return b; return std::nullopt;
}

  std::vector<Block> r; for(auto& b: blocks) if(lower(b.section)==lower(s)) r.push_back(b); return r;
}

  try{ size_t i; double d=std::stod(v,&i); if(i!=v.size()) return false; out=d; return true; } catch(...){ return false; } }

  double d; if(!set_double_on_string(v,d)||d<=0) return false; out=d; return true; }

  std::string l=lower(v); if(l=="true"||l=="1"){out=true;return true;} if(l=="false"||l=="0"){out=false;return true;} return false; }
```

```cpp
namespace hades {
static double mono(){ timespec t; clock_gettime(CLOCK_MONOTONIC,&t); return t.tv_sec + t.tv_nsec/1e9; }
ProcResult run_subprocess(const std::vector<std::string>& argv, const std::string& in,
                          double timeout_s, std::size_t mem_mb){
  int ip[2], op[2], ep[2]; if(pipe(ip)||pipe(op)||pipe(ep)) return {-1,"","",false};
  pid_t pid=fork();
  if(pid==0){                                  // child
    dup2(ip[0],0); dup2(op[1],1); dup2(ep[1],2);
    for(int fd: {ip[0],ip[1],op[0],op[1],ep[0],ep[1]}) close(fd);
    if(mem_mb>0){ rlimit rl{mem_mb*1024*1024, mem_mb*1024*1024}; setrlimit(RLIMIT_AS,&rl); }
    std::vector<char*> a; for(auto& s: argv) a.push_back(const_cast<char*>(s.c_str())); a.push_back(nullptr);
    execvp(a[0], a.data()); _exit(127);
  }
  close(ip[0]); close(op[1]); close(ep[1]);
  if(!in.empty()) (void)!write(ip[1], in.data(), in.size());
  close(ip[1]);                                 // EOF to child stdin
  ProcResult r{0,"","",false};
  pollfd fds[2]={{op[0],POLLIN,0},{ep[0],POLLIN,0}};
  double deadline=mono()+timeout_s; int openfds=2;
  while(openfds>0){
    int ms=(int)((deadline-mono())*1000);
    if(ms<=0){ kill(pid,SIGKILL); r.timed_out=true; break; }
    int n=poll(fds,2,ms); if(n<=0){ if(n==0){ kill(pid,SIGKILL); r.timed_out=true; break; } continue; }
    char buf[4096];
    for(int i=0;i<2;i++){ if(fds[i].revents & (POLLIN|POLLHUP)){
      ssize_t k=read(fds[i].fd,buf,sizeof buf);
      if(k>0){ (i==0?r.out:r.err).append(buf,k); }
      else { fds[i].fd=-1; fds[i].events=0; --openfds; } } }
  }
  close(op[0]); close(ep[0]);
  int st=0; waitpid(pid,&st,0);
  if(!r.timed_out && WIFEXITED(st)) r.code=WEXITSTATUS(st);
  else if(r.timed_out) r.code=-1;
  return r;
}
}  // namespace hades
```

```cpp
namespace hades {
static std::vector<std::string> split_ws(const std::string& s){
  std::vector<std::string> v; std::istringstream is(s); std::string w; while(is>>w) v.push_back(w); return v; }
nlohmann::json mcp_call(const std::string& command, const std::string& tool,
                        const nlohmann::json& args, double timeout_s){
  std::ostringstream in;
  in << nlohmann::json{{"jsonrpc","2.0"},{"id",1},{"method","initialize"},
        {"params",{{"protocolVersion","2024-11-05"},{"capabilities",nlohmann::json::object()},
                   {"clientInfo",{{"name","hades"},{"version","0.1.0"}}}}}}.dump() << "\n";
  in << nlohmann::json{{"jsonrpc","2.0"},{"method","notifications/initialized"}}.dump() << "\n";
  in << nlohmann::json{{"jsonrpc","2.0"},{"id",2},{"method","tools/call"},
        {"params",{{"name",tool},{"arguments",args}}}}.dump() << "\n";
  auto r=run_subprocess(split_ws(command), in.str(), timeout_s);
  if(r.timed_out) return {{"error","mcp server timed out"}};
  std::istringstream out(r.out); std::string line;

    auto j=nlohmann::json::parse(line,nullptr,false);
    if(!j.is_discarded() && j.value("id",0)==2 && j.contains("result")) return j["result"];
  }
  return {{"error","mcp: no result for tools/call"}};
}
```

```cpp
#include "hades/arbiter.h"
#include "hades/blackboard.h"
namespace hades {
void Arbiter::on_attach(Blackboard& bb){
  bb_=&bb;
  bb.subscribe("USER_MESSAGE",[this](const Entry& e){
    history_.push_back({{"role","user"},{"content",e.value}});
    bb_->post("MODE","EXECUTING","arbiter"); start_turn(); });
  bb.subscribe("LLM_RESPONSE",[this](const Entry& e){ on_llm_response(e); });
  bb.subscribe("TOOL_RESULT",[this](const Entry& e){ on_tool_result(e); });
  bb.subscribe("CONFIRM_RESPONSE",[this](const Entry& e){ on_confirm(e); });
}
void Arbiter::start_turn(){
  nlohmann::json tools=nlohmann::json::array();
  for(auto& t: tools_) tools.push_back({{"name",t.name},{"description",t.description},{"schema",t.schema}});
  bb_->post("LLM_REQUEST", {{"messages",history_},{"tools",tools},{"model",model_}}, "arbiter");
}
void Arbiter::on_llm_response(const Entry& e){
  const auto& v=e.value;
  Action act;
  nlohmann::json assistant_msg;
  if(v.contains("tool_call") && !v["tool_call"].is_null()){
    const auto& tc=v["tool_call"];
    act.kind=Action::Kind::ToolCall; act.tool=tc.value("name",""); act.args=tc.value("arguments",nlohmann::json::object());
    act.tool_id=tc.value("id","");
    assistant_msg={{"role","assistant"},{"content",nullptr},{"tool_calls",nlohmann::json::array({
      {{"id",act.tool_id},{"type","function"},
       {"function",{{"name",act.tool},{"arguments",act.args.dump()}}}}})}};
  } else {
    act.kind=Action::Kind::Answer; act.text=v.value("text","");
    assistant_msg={{"role","assistant"},{"content",act.text}};
  }
  bb_->post("NEXT_ACTION", {{"kind",(int)act.kind},{"tool",act.tool},{"text",act.text}}, "arbiter");
  dispatch_or_gate(act, assistant_msg);
}
void Arbiter::dispatch_or_gate(const Action& act, const nlohmann::json& assistant_msg){
  for(auto& o: objectives_){
    if(!o->active(*bb_)) continue;
    auto v=o->veto(*bb_, act);
    if(v.vetoed && v.needs_confirm){
      pending_=nlohmann::json{{"kind",(int)act.kind},{"tool",act.tool},{"args",act.args},{"tool_id",act.tool_id}};
      pending_msg_=assistant_msg;
      bb_->post("CONFIRM_REQUEST", {{"id",act.tool_id},{"prompt",v.reason},
                 {"action",pending_}}, "arbiter");
      return;
    }
    if(v.vetoed){ bb_->post("ASSISTANT_MESSAGE", "[blocked: "+v.reason+"]", "arbiter"); return; }
  }
  if(act.kind==Action::Kind::ToolCall){
    history_.push_back(assistant_msg);
    bb_->post("TOOL_REQUEST", {{"id",act.tool_id},{"tool",act.tool},{"args",act.args}}, "arbiter");
  } else {
    history_.push_back(assistant_msg);
    bb_->post("ASSISTANT_MESSAGE", act.text, "arbiter");
  }
}
void Arbiter::on_tool_result(const Entry& e){
  history_.push_back({{"role","tool"},{"tool_call_id",e.value.value("id","")},
                      {"content",e.value.value("content",nlohmann::json::object()).dump()}});
  start_turn();
}
void Arbiter::on_confirm(const Entry& e){
  if(pending_.is_null()) return;
  if(e.value.value("approved",false)){
    history_.push_back(pending_msg_);
    bb_->post("TOOL_REQUEST", {{"id",pending_["tool_id"]},{"tool",pending_["tool"]},
              {"args",pending_["args"]}}, "arbiter");
  } else bb_->post("ASSISTANT_MESSAGE", "[declined by user]", "arbiter");
  pending_=nullptr; pending_msg_=nullptr;
}
}  // namespace hades
```

```cpp
void ChatModule::on_attach(Blackboard& bb){
```

```cpp
#include "hades/obs/scope.h"
#include <fstream>
#include <iostream>
#include <sstream>
namespace hades {
std::vector<std::string> scope_filter(const std::vector<std::string>& lines, const std::string& prefix){
  if(prefix.empty()) return lines;
  std::vector<std::string> out;
  for(const auto& l: lines){
    std::istringstream s(l); std::string ts,key; std::getline(s,ts,'\t'); std::getline(s,key,'\t');
    if(key.rfind(prefix,0)==0) out.push_back(l);
  }
  return out;
}
}  // namespace hades
int main(int argc, char** argv){
  if(argc<2){ std::cerr<<"usage: hades-scope <log> [KEY_PREFIX]\n"; return 2; }
  std::ifstream f(argv[1]); std::vector<std::string> lines; std::string l;
  while(std::getline(f,l)) lines.push_back(l);
  for(const auto& o: hades::scope_filter(lines, argc>2?argv[2]:"")) std::cout<<o<<"\n";
  return 0;
}
```
